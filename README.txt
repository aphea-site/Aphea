APHEA - English version deployment instructions

1) Create GitHub repository and push this folder
2) Connect repo in Netlify (New site from Git)
3) Publish directory: /
4) Enable Identity and Git Gateway for Netlify CMS
5) Replace GA_MEASUREMENT_ID, META_PIXEL_ID, and Calendly link in index.html
6) Access your live site on Netlify with public link

Enjoy your elegant, feminine, and fully functional Aphea website!
